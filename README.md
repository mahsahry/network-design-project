# Network Design & Simulation (Cisco Packet Tracer)

This repository contains a complete network topology design built in **Cisco Packet Tracer**.

## Skills Demonstrated

- VLAN design and trunk configuration  
- Static and/or dynamic routing  
- IP addressing and subnetting  
- Switch and router configuration  
- End-to-end connectivity testing (ping, traceroute, etc.)  
- Applying basic network security policies (e.g., segmentation, access control)

## How to Open the Project

1. Install **Cisco Packet Tracer**.  
2. Open the file:

   ```text
   network_design_project.pkt
